Актуальная версия: 1.2

Cursor Trail — Windows-приложение на Python и PySide6, которое добавляет на рабочий стол красивый настраиваемый след курсора. Можно менять длину и толщину линии, собирать собственный градиент, включать плавное затухание, прозрачность, свечение, обводку, RGB-анимацию, пиксельный режим и эффект сакуры.

Приложение работает из системного трея, сохраняет настройки и профили, поддерживает светлую/тёмную тему и несколько языков интерфейса.

Скачайте `CursorTrail-Installer-1.2-x64.exe` и запустите его. Это кастомный минималистичный установщик Cursor Trail без CMD-окон: с иконкой приложения, выбором папки, прогрессом установки, ярлыками в меню Пуск/на рабочем столе и записью удаления в настройках Windows.

## Технологии

- Python
- PySide6 / Qt
- PyInstaller
- WinForms GUI installer

## Установка для разработки

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python cursor_trail_qt.py
```

Если PowerShell запрещает активацию окружения, можно запустить напрямую:

```powershell
.\.venv\Scripts\python.exe cursor_trail_qt.py
```

## Сборка приложения

```powershell
python -m PyInstaller "Cursor Trail.spec"
```

После сборки приложение появится в папке:

```text
dist\Cursor Trail\Cursor Trail.exe
```

## Сборка установщика

Установщик собирается из готовой папки `dist\Cursor Trail` в отдельное GUI-приложение. Внутрь установщика встраиваются файлы приложения, иконка, логотип и GUI-деинсталлятор.

```powershell
powershell -ExecutionPolicy Bypass -File installer\build_installer.ps1
```

## Как пользоваться

1. Установите приложение через `CursorTrail-Installer-1.2-x64.exe` или запустите `python cursor_trail_qt.py` из исходников.
2. Откройте настройки через иконку в системном трее.
3. Настройте линию, эффекты, тему, язык и профили.
4. Нажмите `Применить`.
5. Для выхода используйте пункт `Выход` в меню трея.

## Примечания

- Приложение рассчитано на Windows.
- Настройки сохраняются в `settings.json` рядом с исходником или рядом с `.exe`.
- Папки `build`, `dist`, `__pycache__` и виртуальные окружения не коммитятся.
- Если антивирус реагирует на готовую сборку PyInstaller, можно собрать приложение самостоятельно из исходников.
